# Give the Floor to Your Inner Child

Static GitHub Pages site for the MEU Timisoara June 1st campaign.

## Publish

1. Upload the contents of this folder to a GitHub repository.
2. In GitHub, open Settings > Pages.
3. Select the branch and root folder that contain `index.html`.
4. After GitHub Pages publishes, replace `[link]` in the caption with the live URL.

The site is plain HTML, CSS and JavaScript. It has no build step, backend, database or image upload server. Uploaded photos are processed locally in the browser with the Canvas API.
